# Customer-Risk-Prediction

Customer Risk Prediction and Advanced Data Visualization

Phân tích, trực quan hóa dữ liệu hồ sơ khách hàng và dự đoán khách hàng có rủi ro

#  1. Mục tiêu của dự án

Dự án Phân tích Dữ liệu tập trung vào việc Đánh giá Rủi ro Tín dụng cho các tổ chức tài chính. Trọng tâm là việc phát triển một mô hình Học máy (Machine Learning) hiệu suất cao để dự đoán chính xác khả năng vỡ nợ của khách hàng dựa trên dữ liệu hồ sơ. Dự án cũng bao gồm Trực quan hóa Dữ liệu Chuyên sâu (Advanced Data Visualization) để tìm ra mối tương quan rủi ro.

#  2. Các công nghệ được ứng dụng

Python: Xử lý Dữ liệu (Pandas, NumPy)

ML: Xây dựng và kiểm định mô hình Dự đoán Rủi ro

Matplotlib, Seaborn: Trực quan hóa mỗi tương quan giữa các biến

#  3. Mô hình và Kết quả Chính

Trực Quan Hóa: Thực hiện Phân tích Khám phá Dữ liệu trên 6000 hồ sơ (DataFrame có: 250.000 hồ sơ nhưng chênh lệch [0], [1] quá lớn nên đã bỏ bớt), trực quan hóa mối quan hệ giữa các biến tài chính và kết quả rủi ro cuối cùng.

Lựa chọn Mô hình: Xây dựng và so sánh các thuật toán phân loại (Ex: Random Forest, Logistic Regression, XGBoost) để tìm ra mô hình dự đoán rủi ro tốt nhất, đạt Precision 94%, đảm bảo giảm thiểu việc phê duyệt sai các hồ sơ rủi ro cao.

Kỹ thuật Xử lý Dữ liệu: Áp dụng các kỹ thuật tiền xử lý dữ liệu (Pre-processing) như xử lý giá trị thiếu, mã hóa biến phân loại, outliers và chuẩn hóa dữ liệu để tăng cường hiệu suất mô hình.
